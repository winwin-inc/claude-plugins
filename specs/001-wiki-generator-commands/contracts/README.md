## 契约使用指南

### 命令开发者

实现新命令时，必须遵循：

1. **命令接口契约**: 使用统一的命令格式
2. **配置文件契约**: 支持标准配置项
3. **文档模板契约**: 使用预定义模板
4. **质量检查契约**: 返回标准质量结果

### 用户体验者

使用命令时，可以期望：

1. **一致的命令格式**: 所有命令使用相同的参数格式
2. **可预测的输出**: 输出格式符合契约定义
3. **标准化的错误处理**: 错误消息统一格式
4. **可靠的质量检查**: 质量分数计算一致
